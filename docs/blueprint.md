# **App Name**: PharmaBoard Aéroport

## Core Features:

- Secure Access & User Role Management: Implement role-based access control with distinct passwords for the Director and Assistant, securely stored. The Assistant will have permissions to add and edit foundational data like wholesalers and suppliers.
- Intuitive Dashboard Overview: Provide a clean, modern homepage dashboard presenting a concise summary of incoming (sales) and outgoing (orders) finances with immediate graphical trend visualizations for quick comprehension.
- Flexible Data Entry & Import: A secure and optimized section for the Assistant to manually enter daily data or to efficiently upload and process financial data via Excel files, ensuring a streamlined workflow.
- Comprehensive Financial Reporting: Dynamic visual reports with cross-tabulated tables and interactive graphs illustrating detailed financial breakdowns (revenues by type, supplier orders, invoices, returns, credit services, implant consumption) over various periods (daily, fortnightly, monthly, quarterly, semi-annually, annually).
- Insurance Claims & Payout Tracking: Dedicated module to monitor consumption amounts per insurance provider on a monthly or fortnightly basis, including tracking for losses or rejections across all insurances.
- AI-Powered Financial Insights Tool: A generative AI tool designed to analyze complex financial data trends from reports, providing simplified natural language summaries and highlighting significant changes or anomalies to assist the Director in making informed decisions.
- Customizable Data Export to PDF: Enable users to select specific data sets and corresponding graphs to be exported into a professional, well-structured PDF format, suitable for official enterprise reporting.

## Style Guidelines:

- Primary color: A sophisticated, muted blue (#2A89DB) to convey professionalism and trust, chosen to be impactful against a light background.
- Background color: An extremely light and desaturated cool-toned grey (#F0F5F8), creating a clean and spacious feel for optimal readability, especially for an elderly user.
- Accent color: A vibrant turquoise (#19E5E5), analogous to the primary color but with high contrast, used strategically for interactive elements and call-to-actions to draw attention effectively.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, slightly technical yet clean aesthetic, fitting for data visualization. Body text font: 'Inter' (sans-serif) for its excellent legibility and versatility, ideal for detailed financial reports and daily use.
- Utilize modern, easily recognizable vector icons that clearly represent financial data, reporting, pharmacy operations, and user actions, ensuring intuitiveness for all users.
- Employ a responsive, clean, and organized grid-based layout with generous white space. Key information will be presented with large, clear labels and interactive elements to ensure ease of navigation and comprehension.
- Incorporate subtle and smooth transitions for data filtering, chart loading, and page navigation to create a fluid user experience without causing distraction or delays.
